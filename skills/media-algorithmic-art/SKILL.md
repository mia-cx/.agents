# Algorithmic Art Workflow

Treat this file as a router. The work happens in two phases: define the generative philosophy, then implement only the guides needed for the artifact.

## Workflow

1. Start with the concept and create the algorithmic philosophy.
   - Read `references/philosophy.md` for the philosophy-writing process, conceptual-seed guidance, and the standards for subtle thematic references.
2. Only after the philosophy is solid, read `references/implementation.md` for the p5.js implementation rules, template usage, UI controls, and artifact packaging.
3. Before writing HTML, read `templates/viewer.html` exactly as the implementation guide requires.
4. Keep the artwork original, generative, and reproducible through seeded randomness.

## Rules

- Do not jump straight to implementation without first establishing the philosophy.
- Keep the template structure and Anthropic-branded shell intact where the implementation guide says it is fixed.
- Favor original generative systems over style mimicry or static illustration tricks.
