# Anthropic Brand Styling

Treat this file as a router. Read the brand-system guide when the artifact should specifically match Anthropic’s visual identity.

## Workflow

1. Read `references/brand-system.md` for colors, typography, accent usage, and technical application notes.
2. Apply the brand system to the target artifact without drifting into generic theming.
3. Preserve readability and hierarchy while aligning the work to Anthropic’s look and feel.

## Rules

- Use this skill only for Anthropic-branded outputs, not generic styling.
- Keep the main file as a routing layer; detailed brand tokens live in the reference.
- Prefer consistent application of the existing brand system over improvising new motifs.
